# Backend module - online bookshop

#### Актуальная ветка 
- main

#### Руководство для новичков
- mvn clean install -DskipTests=true
- run/debug BookApplication

#### Требования для сборки
- Java 17(JDK)
- Maven v3.2.x и выше

#### Язык/Фреймворк
- Java 17 (JDK)
- Spring Boot 3.1.3

#### База данных
- PostgreSQL

#### Тип приложения / Используемый контейнер
- Executable JAR
